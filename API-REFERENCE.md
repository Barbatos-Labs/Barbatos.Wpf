# Barbatos.Wpf API Reference

Barbatos.Wpf ships as several independent packages, each with its own full API
reference:

- **[Barbatos.Wpf.Core API Reference](https://github.com/Barbatos-Labs/Barbatos.Wpf/blob/main/src/Barbatos.Wpf.Core/API-REFERENCE.md)** -
  `Barbatos.Wpf.Hosting`, `Barbatos.Wpf.LifecycleEvents`, `Barbatos.Wpf.Dispatching`,
  `Barbatos.Wpf.ApplicationModel`, `Barbatos.Wpf.Devices`, `Barbatos.Wpf.Networking`,
  `Barbatos.Wpf.Storage`, `Barbatos.Wpf.Tray`, `Barbatos.Wpf.Power`,
  `Barbatos.Wpf.Startup`, `Barbatos.Wpf.Notifications`, `Barbatos.Wpf.SingleInstance`,
  `Barbatos.Wpf.Dialogs`.
- **[Barbatos.Wpf.Aquarius API Reference](https://github.com/Barbatos-Labs/Barbatos.Wpf/blob/main/src/Barbatos.Wpf.Aquarius/API-REFERENCE.md)** -
  `Barbatos.Wpf.Aquarius.Reactivity`, `Barbatos.Wpf.Aquarius.Composition`,
  `Barbatos.Wpf.Aquarius.Xaml`, `Barbatos.Wpf.Aquarius.Animation`.

See the [root README](https://github.com/Barbatos-Labs/Barbatos.Wpf#readme) for an overview of both packages.
